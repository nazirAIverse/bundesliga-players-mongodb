# Bundesliga Players Data Project

## Overview
This project processes Bundesliga player data using Pandas and stores it in MongoDB.

## Technologies
- Python
- Pandas
- MongoDB
- PyMongo
- MongoEngine
- Docker

## Features
- Data cleaning
- Batch insertion into MongoDB
- CRUD operations
- ODM queries using MongoEngine
